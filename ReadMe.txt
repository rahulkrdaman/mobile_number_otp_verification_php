============ Instruction ============
1. Create a database called "yourdatabase" at phpMyAdmin and import the "SQL/mobile_numbers.sql" file into this database.
2. Open the "DB.class.php" file ==> Specify the database hostname ($dbHost), username ($dbUsername), password ($dbPassword), and name ($dbName) as per your MySQL credentials.
3. In the index.php file, specify your SMS gateway username and API Key in sendSMS() function.
4. Open the "index.php" file on the browser and test the OTP SMS functionality.
